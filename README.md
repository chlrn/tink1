# Мультиязычная игра для изучения слов

## Описание проекта

Этот проект представляет собой десктопное приложение на Java с использованием JavaFX, разработанное в формате игры для обучения языкам. Игра фокусируется на выборе правильного перевода слова на трех языках: английском, французском и немецком. Пользователь может выбрать уровень сложности (от A1 до C2), что позволяет адаптировать обучение в зависимости от знаний игрока.

## Функционал

- **Выбор языка**: Игра предлагает пользователю перевести слова на английский, французский или немецкий языки.
- **Выбор уровня**: Доступны уровни сложности от A1 до C2, что позволяет игрокам адаптировать игру под свои знания.
  - При правильном ответе игроку показывается радостная картинка, и очки суммируются.
  - При неверном ответе появляется грустная картинка, и игрок продолжает уровень.
- **Количество слов**: В каждом уровне содержится 15 слов для перевода.
- **API Яндекс Переводчика**: Используется для получения переводов слов через JSON-запросы.
- **Docker**: Приложение упаковано в контейнер Docker, что позволяет легко развернуть проект и управлять его зависимостями.

## Установка и запуск

### Требования

- Java JDK 17 или выше
- Docker

## Установка

### 1. Клонирование репозитория

Сначала клонируйте репозиторий на ваш локальный компьютер:

```sh
git clone https://github.com/chlrn/translation-service.git
cd translation-service
```
### 2. Сборка приложения
Соберите приложение с помощью Maven.

```sh
./mvnw package
```
### 3. Создание Docker образа
Соберите Docker образ из Dockerfile:
```sh
docker build -t translation-service .
```

### 4. Запуск Docker контейнера
Запустите контейнер с приложением
```sh
docker run -p 8080:8080 translation-service
```

### Запуск приложения

После запуска контейнера откройте браузер и перейдите по адресу `http://localhost:8080` для доступа к приложению.

## Использование

- Запустите приложение.
- Выберите желаемый язык и уровень сложности.
- Игра начнется с представления слова, выберите правильный перевод из предложенных вариантов.
- Смотрите на изображения, которые отображаются после каждого ответа, и следите за накоплением очков.
